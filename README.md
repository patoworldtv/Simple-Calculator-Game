# Simple-Calculator-Game
The game generates random arithmetic problems (addition, subtraction, multiplication, division), asks the player to solve them, checks the answer, and awards points. The player can choose difficulty levels and play multiple rounds. This project demonstrates input handling, functions, randomness, loops, and basic game logic.

# Gameplay
Choose a difficulty level

Solve randomly generated math problems

Get immediate feedback

Earn points for correct answers

Play as many rounds as you want

# Features
Random arithmetic problem generator

Four operations: +, –, ×, ÷

Difficulty levels (Easy, Medium, Hard)

Score tracking

Clean and readable code structure
